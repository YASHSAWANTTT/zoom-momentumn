HOW IT WORKS

From Install to Impact in Minutes

No complex configuration. No learning curve. Just open Zoom and teach.

01

Install the Zoom App

Add Zoom Momentum from the Zoom Marketplace. It loads as an in-meeting side panel — zero setup for students.

02

Start Your Lecture

The Warm-Up Arena auto-launches trivia. Live Anchor begins building a shared timeline from the transcript stream.

03

Engage in Real Time

Professors trigger Pulse check-ins at any moment. Students bookmark confusion points. Everything syncs live.

04

Learn After Class

Recovery Agent generates personalized study packs from bookmarks. Students get targeted explanations and practice.
